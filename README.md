# Order Book

A real-time order book service for submitting, matching, and broadcasting trades. Built with Java and Spring Boot.

## Features

- Accepts buy and sell orders through a REST API
- Matches compatible orders and generates trades
- Maintains an in-memory order book
- Broadcasts order book updates via WebSocket
